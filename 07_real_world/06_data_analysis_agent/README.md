# Example 06: Data Analysis Agent

An agent that analyzes data and generates insights.

## Features

- CSV/JSON data loading
- Statistical analysis
- Trend identification
- Visualization suggestions
- Report generation

## Architecture

```
Data Input → Parse → Analyze → Insights → Report
```

## Key Concepts

- **Structured Output**: Consistent analysis format
- **Tools**: Data processing functions
- **Instructions**: Analysis methodology

## Run the Example

```bash
python main.py --data sales.csv "What are the key trends?"
```
