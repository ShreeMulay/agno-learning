# CSAT Analyzer

**Example #024** | Category: `business/customer_success`

Analyzes customer satisfaction surveys to extract themes, trends, and actionable insights.

## Usage

```bash
python main.py --responses "Survey data" --period "last 30 days"
```

## Features
- Score distribution analysis
- Theme extraction (positive/negative)
- Trend identification
- Top performer recognition
- Priority issue flagging
- Executive summary generation
