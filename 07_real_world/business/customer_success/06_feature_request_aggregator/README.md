# Feature Request Aggregator

**Example #026** | Category: `business/customer_success`

Clusters and prioritizes feature requests using RICE framework for roadmap planning.

## Usage

```bash
python main.py --requests "Feature request data"
```

## Features
- Request clustering by theme
- RICE prioritization scoring
- Trend analysis
- Competitive gap identification
- Churn risk flagging
- Roadmap recommendations
