# Ticket Router

**Example #021** | Category: `business/customer_success`

Automatically classifies, prioritizes, and routes support tickets to the appropriate team.

## Usage

```bash
python main.py --subject "Login issue" --body "Can't access account" --tier enterprise
```

## Features
- Multi-category classification
- Priority and urgency scoring
- SLA-based routing
- Information extraction
- KB article recommendations
- Response template suggestions
