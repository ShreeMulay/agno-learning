# Ad Copy Generator

**Example #013** | Category: `business/marketing`

Creates high-converting ad copy with A/B test variants for Google, Facebook, LinkedIn, and Twitter.

## Usage

```bash
python main.py --product "Your Product" --platform google --goal signups
```

## Features
- Multi-platform support with character limit compliance
- Multiple copywriting frameworks (AIDA, PAS, BAB)
- A/B test variants generation
- Targeting recommendations
- Landing page suggestions
