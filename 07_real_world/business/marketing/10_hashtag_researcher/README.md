# Hashtag Researcher

**Example #020** | Category: `business/marketing`

Researches and recommends optimal hashtags for social media with trending analysis and curated sets.

## Usage

```bash
python main.py --topic "machine learning" --platform instagram --niche "tech"
```

## Features
- Trending hashtag discovery
- Competition level analysis
- Relevance scoring
- Curated hashtag sets (30-30-30-10 rule)
- Banned hashtag detection
- Copy-paste ready output
