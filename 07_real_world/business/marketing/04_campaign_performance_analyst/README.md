# Campaign Performance Analyst

**Example #014** | Category: `business/marketing`

Analyzes cross-channel marketing campaign performance with ROI calculations, attribution insights, and budget optimization recommendations.

## Usage

```bash
python main.py --budget 10000 --goal conversions
```

## Features
- Multi-channel performance breakdown
- ROI, ROAS, CPA calculations
- Attribution analysis
- Budget reallocation recommendations
- Anomaly detection
- Actionable next steps
