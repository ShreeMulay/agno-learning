## Common Patterns

- **Tools + Structured Output**: CRM integration, data extraction
- **Knowledge + Memory**: Policy Q&A, customer context
- **Teams + Workflows**: Multi-department coordination
- **Structured Output**: Reports, dashboards, metrics

## Getting Started

1. Choose a subcategory that matches your use case
2. Review the examples within that subcategory
3. Start with the simplest example, then build complexity
4. Customize configuration for your specific needs

## Prerequisites

Many business agents integrate with external systems:
- CRM (Salesforce, HubSpot)
- Communication (Slack, Email)
- Finance (QuickBooks, Stripe)
- HR (Workday, BambooHR)

Set up API keys in `.env` for full functionality. All examples include demo mode for testing without credentials.
