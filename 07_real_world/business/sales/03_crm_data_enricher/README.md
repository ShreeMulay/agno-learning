# CRM Data Enricher

Automatically enriches CRM contact records with company intelligence.

## Use Cases

- Keeping CRM data fresh and accurate
- Pre-call research automation
- Account-based marketing data prep
- Lead scoring input

## Patterns Used

- **Tools**: DuckDuckGo for comprehensive research
- **Structured Output**: Rich schema with nested tech stack

## Running

```bash
python main.py --company "Notion" --contact "Ivan Zhao"
```

## Output Fields

- Company basics (industry, size, HQ, website)
- Funding status and recent rounds
- Technology stack details
- Recent news and developments
- Sales-ready pain points and talking points
