# Proposal Generator

Creates customized sales proposals with company research and personalization.

## Use Cases

- RFP responses
- Enterprise sales proposals
- Renewal proposals with usage data
- Upsell/expansion proposals

## Running

```bash
python main.py --company "Notion" --contact "Ivan Zhao" --product "AI Platform" --price 75000
```

## Output

- Executive summary tailored to prospect
- Identified challenges from research
- Value propositions mapped to challenges
- Itemized pricing
- Implementation timeline
- Clear next steps
