# Example 05: Blog Post Generator

A workflow that generates blog posts with research, writing, and editing.

## Features

- Topic research
- Outline generation
- Content writing
- SEO optimization
- Final editing

## Architecture

```
Topic → Research Step → Outline Step → Write Step → Edit Step → Final Post
```

## Key Concepts

- **Workflows**: Multi-step content generation
- **State**: Pass context between steps
- **Structured Output**: Consistent formats

## Run the Example

```bash
python main.py "The future of renewable energy"
```
